Plugins that will make your work more comfortable: Плагины, которые сделают вашу работу комфортнее:
Emmet Помогает ускорить вёрстку. С помощью ! и tab можно создать базовую структуру кода за 1 секунду, либо быстро создать вложенные теги.
VS code Icons Файлы, и папки, будут обладать иконками в зависимости от кода прописанного внутри них.
Live Server Локальный сервер для своего проекта.
Bracket Pair Colorizer Добавляет подсветку для фигурных скобок и тем самым позволяет вам быстро ориентироваться среди них
Auto rename tag Если тег повторяется, при замене одного с помощью плагина, все другие будут переименованы автоматически.
Indent Rainbow За счет расширения вы получаете цветовое оформление для отступов внутри проекта.
GitLens Предоставляет вам полную информацию относительно репозитория вашего проекта, а также всех комитов и комментариев что были внесены и оставлены.
Faker Создать фейковую информацию и сразу же подставить её внутрь некой переменной.
